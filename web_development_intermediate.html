Web development napredno

Kurs za djecu koja žele naučiti izrađivati moderne statičke i dinamičke internet stranice, back-end programiranje, baze podataka neophodne za back-end programiranje. Nakon završenog kursa djeca će imati potrebno predznanje za Full-stack razvoj.

O kursu

Završni kurs iz oblasti web developmenta tokom kojeg ćemo naučiti koristiti Bootstrap 5, API, hosting servise, noSQL baze podataka. Tokom kursa imaćemo i uvod u programski jezik Python kako bi djeca mogla raditi sa back-end modulom Flask.

Potrebno predznanje

Rad na računaru, engleski jezik, osnove web razvoja.

Program kursa

Web programiranje kroz praktičan rad, projekti, vježbe, provjera znanja kroz izazove.

Tokom kursa učićemo razne šablone potrebne za web razvoj, poput flask šablona, jinja šablona, kao i koncepte i rad sa programskim jezikom Javascript.
