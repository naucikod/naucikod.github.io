Applied Computer Vision & Edge AI Program

Instructor-led, project-based curriculum (ages 12–18)

Program Overview

This is a structured, semester-style program designed to take students from Python fundamentals to applied Computer Vision, Deep Learning, and Edge AI deployment.

Learning Objectives

  • Build Computer Vision pipelines using OpenCV
  • Create and label datasets
  • Train deep learning models
  • Deploy AI to edge devices
  • Integrate AI with robotics systems

Course Structure

Phase 1 – Python & Image Processing

Phase 2 – Deep Learning

Phase 3 – Edge AI Deployment

Phase 4 – Robotics Integration
